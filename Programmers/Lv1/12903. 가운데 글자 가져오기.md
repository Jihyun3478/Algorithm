#### ✔️ Problem
프로그래머스 모바일은 개인정보 보호를 위해 고지서를 보낼 때 고객들의 전화번호의 일부를 가립니다.
전화번호가 문자열 phone_number로 주어졌을 때, `전화번호의 뒷 4자리를 제외한 나머지 숫자를 전부 *으로 가린 문자열`을 리턴하는 함수, solution을 완성해주세요.



#### ✔️ Code
```java
class Solution {
    public String solution(String s) {
        String answer = "";
        
        if(s.length() % 2 == 0) {
            answer = s.substring(s.length()/2-1, s.length()/2+1);
        }
        else {
            answer = s.substring(s.length()/2, s.length()/2+1);
        }
        return answer;
    }
}
```
