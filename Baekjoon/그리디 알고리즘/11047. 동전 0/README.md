### [BOJ] 11047. 동전 0

---

#### ✔️ Problem URL
[동전 0](https://www.acmicpc.net/problem/11047)

#### ✔️ Problem
준규가 가지고 있는 동전은 총 N 종류이고, 각각의 동전을 매우 많이 가지고 있다.

동전을 적절히 사용해서 그 가치의 합을 K로 만들려고 한다. 이때 필요한 동전 개수의 최솟값을 구하는 프로그램을 작성하시오.
