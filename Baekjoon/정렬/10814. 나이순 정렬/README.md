### [BOJ] 10814. 나이순 정렬

---

#### ✔️ Problem URL
[나이순 정렬](https://www.acmicpc.net/problem/10814)

#### ✔️ Problem
온라인 저지에 가입한 사람들의 나이와 이름이 가입한 순서대로 주어진다.
이때, 회원들을 나이가 증가하는 순으로, 나이가 같으면 먼저 가입한 사람이 앞에 오는 순서로 정렬하는 프로그램을 작성하시오.

💡 본 문제는 값이 같은 원소의 전후 관계가 바뀌지 않는 stable sort가 적용된다.
